How to Export from Your Broker

Trade Nation: Go to History → Click Export → Select date range → Download CSV
TD365: Go to Account → Transaction History → Export to CSV
Tips:
  • Duplicate transactions are automatically skipped
  • Make sure to select the correct account and format
  • Files should include Transaction Date, Action, P/L columns
